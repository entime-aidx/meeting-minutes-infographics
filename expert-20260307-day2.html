🔄 実技：GAS ⇄ Base 双方向連携

① GAS → Base

スプレッドシート → GAS実行 → Webhook URL → Baseレコード追加

② Base → GAS

Baseボタン → HTTP POST → GASデプロイURL → シート書き込み

📦 JSON と GAS

JSON＝HTTP通信時のデータ形式。キーとバリュー。文字列はダブルクォーテーション、数字はそのまま。
GAS＝Google Apps Script。コンテナバインド（スプシ紐づき）とスタンドアロン（ドライブ単体）の2種類。

❓ 質疑の要点

  • 仕事での活用：各APIを調べ、Lark・Baseと連携。kintone・Salesforce等は外部連携知識が必須
  • 自動化なしで可能？Lark APIを直接叩く場合はGASだけで完結
  • AIでコード生成：処理の流れ・Webhookの有無を明確に伝えると精度が上がる
  • エラー調査：手がかり（ステータスコード・ログ）を掴んでからAIに投げる
